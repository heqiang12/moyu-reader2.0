# 摸鱼阅读器 2.0 (Moyu Reader 2.0)

> **一款专为办公环境设计的独立桌面“摸鱼”阅读器。**  
> 核心卖点：**极致轻量（<10MB）**、**极低内存（<50MB）**、**无边框自由拖拽**、**窗口伪装变形** 与 **全局老板键**。

---

## 📖 产品理念与核心场景

在办公环境下，传统的电子书阅读器（如网页端、大型桌面客户端）界面特征太明显，容易被同事或领导一眼识破。

**摸鱼阅读器 2.0** 通过将**小说文本内容伪装成正常工作界面**，并结合**高自由度的悬浮窗拖拽与变形能力**，完美解决了这个问题。

### 核心功能矩阵

*   **📚 智能书架**：支持本地 TXT 导入，自动识别 UTF-8 / GBK 编码，自动提取章节目录，记录阅读进度。
*   **🥷 2 种阅读模式**：
    1.  **正常模式**：标准极简电子书排版，适合专注阅读。
    2.  **隐蔽模式（核心卖点）**：阅读时自动开启，窗口缩为极窄单行文本条，可自定义背景色、背景透明度和区域样式，支持自由拖拽到屏幕任意位置，鼠标滚轮翻行，完美融入桌面环境不引人注意。
*   **⌨️ 全局老板键**：通过 Tauri 全局快捷键插件注册系统级热键（默认 `Ctrl + ~`）。无论当前窗口是否聚焦，按下即可瞬时隐藏至系统托盘，任务栏不留痕迹。
*   **🖱️ 无缝拖动与无边框**：去除原生系统窗口边框（标题栏），界面各区域可通过拖拽自由移动，双击或快捷键快速调整大小。

## 🛠️ 技术选型与架构

### 技术栈

*   **桌面外壳**：**Tauri v2** (Rust) — 抛弃了 Electron 臃肿的 Chromium 内核，采用系统原生 WebView（Windows 上为 WebView2），使打包体积控制在 **10MB 以内**，运行内存 **< 50MB**。
*   **前端核心**：**Vite + 原生 HTML5/CSS3/Vanilla JS** — 零框架运行时开销，保证最快的启动速度和最灵敏的窗口变形响应。
*   **数据存储**：**localStorage + 浏览器 IndexedDB** — 用于高频的书籍元数据、阅读进度与用户配置存储，大文件文本则通过 Tauri 异步读取本地文件。

## 🥷 阅读模式详细视觉规范

### 1. 正常模式 (Normal Mode)
*   **窗口尺寸**：可自由拉伸，默认 800 × 600 px
*   **视觉**：标准极简电子书排版，清晰易读，适合专注阅读场景。
*   **操作**：方向键 / 鼠标滚轮 / 点击翻页，支持章节跳转。

### 2. 隐蔽模式 (Stealth Mode)
*   **窗口尺寸**：默认 800 × 32 px，宽度可调节
*   **边框**：无边框，无任何按钮，极简到像一条系统状态提示
*   **窗口位置**：默认位于桌面最下方，可自由拖拽到屏幕任意位置
*   **视觉自定义**：
    *   背景色支持任意颜色选择（含透明度调节，`rgba` 全范围可调）
    *   字体颜色、字号可自定义
    *   整体可设为半透明，完美融入任何桌面壁纸
*   **操作**：
    *   鼠标滚轮向上/向下翻行
    *   鼠标拖拽移动位置
    *   双击或快捷键切换回正常模式
*   **特性**：阅读时自动开启，无需手动切换

## ⌨️ 老板键与防窥交互

为了实现最强防窥，应用将集成以下机制：

1.  **系统全局热键**：利用 Tauri Rust 端的全局热键监听，当用户按下自定义热键（默认 `Ctrl + ~`）时：
    *   窗口**瞬间隐藏**，在任务栏彻底消失。
    *   如果开启了“快速闪避”伪装，隐藏的同时可以自动调用默认浏览器打开一个预设的工作页面（如百度、谷歌、公司内网门户）。
2.  **托盘图标低调化**：托盘图标采用类似系统网络、音量或防病毒软件的低调图标（如灰色小盾牌或齿轮），右键菜单项也进行合理伪装（如“同步服务”、“系统检测”等）。
3.  **自动失焦隐藏**：支持配置“当鼠标离开窗口范围 X 秒后自动半透明或缩回单行”以及“当窗口失去焦点时自动隐藏”。

## 📅 路线图与开发规划

### 第一阶段：准备工作与环境搭建 (MVP)
*   配置 Rust 及 Windows 编译链。
*   使用 Tauri CLI 初始化项目。
*   设计原生前端界面系统框架。

### 第二阶段：书架与阅读核心
*   实现本地文本文件解析与编码自动检测。
*   开发物理分页引擎与阅读进度记录。
*   完成自定义标题栏和基本的窗口拖动功能。

### 第三阶段：隐蔽模式与变形系统
*   开发隐蔽模式的单行文本渲染与背景自定义功能。
*   利用 Tauri 窗口 API 实现窗口大小、置顶、透明度变换。
*   实现隐蔽模式自由拖拽与位置记忆。
*   集成 Rust 端全局快捷键与低调系统托盘。

### 第四阶段：配置与打包优化
*   提供完善的设置面板。
*   支持自定义正则解析小说章节。
*   编译打包生成单文件 Windows 绿化版 `.exe`。
